
Система работает локально на устройстве и в конце дня выдаёт краткий отчёт о результатах анализа сообщений пользователя.

Еще пришла идея - включить оценку сообщений пользователей

## 📊 5 МЕТРИК ПО ИТОГУ СЕССИИ

### Метрика 1: Полнота детекции (Recall)

Доля реально опасных сообщений, которые система поймала через регулярные выражения.

```
Recall = N_detected_danger / N_true_danger

где:
  N_true_danger = количество сообщений, помеченных как реально опасные через анализ промптом
  N_detected_danger = из них сколько система выловила (сработал регулярный выраж.)
```

**Целевое значение:** ≥ 0.90 (система должна ловить минимум 90% реальных опасностей)

**Интерпретация:**
- 0.90+ = отлично, почти ничего не пропускаем
- 0.80–0.90 = приемлемо, но есть небольшие пропуски
- < 0.80 = есть проблемы, нужна доработка регулярок

### Метрика 2: Доля ложных тревог (False Positive Rate, FPR)

Как часто система видит опасность там, где её нет.

```
FalsePositiveRate = N_false_positive / N_flagged_danger

где:
  N_flagged_danger = количество сообщений, помеченных системой как опасные (YELLOW/ORANGE/RED)
  N_false_positive = из них промптом анализа или пользователем отмечено как "неправильно, это не опасно"
```

**Целевое значение:** ≤ 0.20 (не более 20% ложных срабатываний)

**Интерпретация:**
- ≤ 0.10 = отлично, система точная
- 0.10–0.20 = приемлемо
- > 0.20 = высокий FPR, нужно снизить чувствительность или переобучить

### Метрика 3: Доля опасных сообщений за день (Risk Share)

Какой процент всех сообщений за день помечен как опасный.

```
RiskShare = N_flagged_danger / N_total

где:
  N_total = всего сообщений за день
  N_flagged_danger = сообщения в зонах YELLOW/ORANGE/RED
```

**Интерпретация:**
- ≤ 1% = обычный день, редкие тревожные мысли
- 1–5% = повышенное напряжение, есть мысли, которые нужно отслеживать
- > 5% = тяжёлый день, много тревожного контента

### Метрика 4: Распределение по зонам риска

Сколько сообщений попало в каждую зону риска.

```
P_GREEN = N_green / N_total
P_YELLOW = N_yellow / N_total
P_ORANGE = N_orange / N_total
P_RED = N_red / N_total
```

**Желаемая картина:**
- P_RED ~ 0–1% (RED должна быть редкой)
- P_ORANGE ~ 1–3% (сильные тревоги, но контролируемые)
- P_YELLOW ~ 3–10% (буфер для наблюдения)
- P_GREEN ~ 85–95% (основная часть, нормальное общение)

### Метрика 5: Распределение по классам опасности

Какого типа риски доминировали в течение дня.

```
P_class(c) = N_c / (N_self_harm + N_psychosis + N_delusion + N_obsession + N_anthro)

где c ∈ {SELF_HARM, PSYCHOSIS, DELUSION, OBSESSION, ANTHROPOMORPHIZATION}
```

**Вывод в конце дня:**
```
Классы опасности, выявленные сегодня:
├─ SELF_HARM: 40% (6 сообщений)
├─ PSYCHOSIS: 15% (2 сообщения)
├─ DELUSION: 10% (1 сообщение)
├─ OBSESSION: 25% (4 сообщения)
└─ ANTHROPOMORPHIZATION: 10% (1 сообщение)
```

## 🚨 5 КЛАССОВ ОПАСНОГО ПОВЕДЕНИЯ

Каждый класс имеет **три зоны риска**, разделённые регулярными выражениями:
- **HIGH** → RED zone
- **MEDIUM** → ORANGE zone
- **LOW** → YELLOW zone
Если сообщение распознано как неопасное, то это зеленая зона

```python
    # Проверяем регулярки
```

## 🔧 БЫСТРАЯ ИНТЕГРАЦИЯ

### Python: Минимальный скрипт анализа

```python
import re
from collections import defaultdict

class LocalRiskAnalyzer:
    def __init__(self):
        self.regexes = {
            'SELF_HARM': {
                'HIGH': [r'(?i)(хочу\s+(умереть|покончить))\b', r'(?i)(want\s+to\s+die)\b'],
                'MEDIUM': [r'(?i)(не\s+хочу\s+жить)\b', r"(?i)(don't\s+want\s+to\s+live)\b"],
                'LOW': [r'(?i)(депрессия|грустно)\b', r'(?i)(depressed|sad)\b'],
            },
            # ... остальные классы ...
        }
    
    def analyze_message(self, message: str) -> dict:
        """Анализирует одно сообщение."""
        result = {'zone': 'GREEN', 'classes': {}}
        max_level = 0
        level_map = {'HIGH': 3, 'MEDIUM': 2, 'LOW': 1}
        
        for class_name, levels in self.regexes.items():
            for level, patterns in levels.items():
                if any(re.search(p, message) for p in patterns):
                    result['classes'][class_name] = level
                    max_level = max(max_level, level_map[level])
        
        # Определяем зону по максимальному уровню
        zone_map = {3: 'RED', 2: 'ORANGE', 1: 'YELLOW', 0: 'GREEN'}
        result['zone'] = zone_map[max_level]
        return result
    
    def daily_report(self, messages: list, ground_truth: list):
        """Собирает дневной отчёт."""
        results = [self.analyze_message(msg) for msg in messages]
        
        # Считаем метрики
        detected = sum(1 for i, r in enumerate(results) if r['zone'] != 'GREEN' and ground_truth[i])
        total_danger = sum(ground_truth)
        recall = detected / total_danger if total_danger > 0 else 1.0
        
        flagged = sum(1 for r in results if r['zone'] != 'GREEN')
        false_pos = sum(1 for i, r in enumerate(results) if r['zone'] != 'GREEN' and not ground_truth[i])
        fpr = false_pos / flagged if flagged > 0 else 0.0
        
        print(f"Recall: {recall:.1%}, FPR: {fpr:.1%}, Risk Share: {flagged/len(messages):.1%}")
```
